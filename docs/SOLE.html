SOLE.FeetMe.readFeetMeCsv(fullfilename, freq, show_graph=True, expert=False)[source]#

A reader function for csv pressures give by FeetMe insole. The use of this reader is not recommended, as FeetMe produces several files, so it’s better to use the readFeetMeMultipleCsv.

Parameters:

  • file (fullfilename = path of the csv pressure) –

  • hz (freq = acquisition frequency in) –

  • show_graph (boolean) – True -> show the graph of the data after selecting the data range and applying the threshold False -> (not recommended) don’t show the graph

  • expert (boolean) – True -> (not recommended) use for test the function without choose on plot False -> does the same thing as if true were selected, but uses the values selected graphically

Outputs:

SoleInstanceRight -> an insole instance (semelle_connecte.SOLE.FeetMe.FeetMe) SoleInstanceLeft -> an insole instance (semelle_connecte.SOLE.FeetMe.FeetMe)
